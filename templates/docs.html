Coats of Arms API Documentation

This page documents the REST API for managing coats of arms. Use the API key for any write operations.

Base URL

http://localhost:5000/api

Authentication

Include your API key for POST, PUT, and DELETE requests:

  • Header: X-API-Key: <your_api_key>
  • Query param: ?api_key=<your_api_key>
Only the user who created an item can edit or delete it.

Endpoints

1. List All Items

GET /items

Returns an array of all coats of arms.

2. Get Single Item

GET /items/<id>

Returns a single coat of arms by ID.

{
  "id": 1,
  "name": "Example Coat",
  "motto_latin": "Virtus et Honos",
  "motto_english": "Virtue and Honor",
  "motto_other": {"fr": "Vertu et Honneur"},
  "colors": ["red", "gold"],
  "symbols": ["lion", "crown"],
  "shield_shape": "heater",
  "created_at": "2026-05-06",
  "designer": "Jane Doe",
  "image": "https://example.com/image.png",
  "description": "A sample coat of arms",
  "usage_official_documents": 1,
  "usage_flags": 0,
  "usage_seal": 0
}

3. Create Item

POST /items (API key required)

Request JSON body example:

{
  "name": "New Coat of Arms",
  "motto_latin": "Virtus et Honos",
  "motto_english": "Virtue and Honor",
  "motto_other": {"fr": "Vertu et Honneur"},
  "colors": ["red", "blue"],
  "symbols": ["lion", "star"],
  "shield_shape": "round",
  "created_at": "2026-05-06",
  "designer": "John Doe",
  "image": "https://example.com/image.png",
  "description": "Sample description",
  "usage_official_documents": 1,
  "usage_flags": 0,
  "usage_seal": 0
}

4. Update Item

PUT /items/<id> (API key required)

Only the creator can update. Example body:

{
  "name": "Updated Coat",
  "motto_latin": "Fortis et Fidelis",
  "colors": ["green", "gold"]
}

5. Delete Item

DELETE /items/<id> (API key required)

Only the creator can delete. Response:

{
  "status": "deleted"
}

Notes

  • colors and symbols are JSON arrays.
  • motto_other is a JSON object for multiple languages.
  • usage_official_documents, usage_flags, usage_seal are 0 or 1.
